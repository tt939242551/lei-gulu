# 安装
#### 推荐使用yarn安装
```
yarn add lei-gulu-1
```
```
npm install lei-gulu-1
```

## 引入 gulu
```
import {Button,Icon} from 'lei-gulu-test-1'
import 'lei-gulu-test-1/dist/index.css'

export default {
  name: 'app',
  components: {
    'g-button': Button,
    'g-icon': Icon
  }
}
```
